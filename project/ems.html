Welcome to Employee Mangaement System project details -2023

PHP project that can make employer work convenient and easy to manage all the employees within the company.

Technologies used

Developed the frontend using React.js, focusing on creating reusable components and ensuring a responsive design. Implemented the backend with PHP and Database used is MySQL.

  • HTML
  • CSS
  • JavaScript
  • PHP
  • MySQL Database
  • AdminLTE Template
  • Bootstrap CSS Framework

    • Key Features

    • Each employee can update their profile and password on the system on the employee side of the system.
    • They can only create one leave application on the system which means if they already created a leave application and still in pending status, employee can't add another application until the pending one is approved or denied.
